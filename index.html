I work with small and medium-sized businesses, design teams and agencies, building fast, responsive and utilizable website front-ends, with strong focus on HTML5, CSS, JavaScript, responsive design and performance.

If you’re a designer or a design agency looking for someone to implement your designs in a responsive, performant manner, and think you can benefit from my skill set, then I’m happy to hear from you.

What I can help you with:

  • Front-end development: Working with UI/UX designers to convert designs into responsive, performant and standards-compliant websites, using HTML5, CSS and JavaScript.
    • Responsive Design and grid frameworks (Bootstrap and Foundation).
    • CSS pre-processor including LESS and SASS.
    • JavaScript MVC / MVW frameworks - AngularJS.
    • RequireJS for file and module loader.
    • Bower for dependency management.

What I can’t help you with:

Knowledge-Based Intranet Portal

Project Description
Proposed a newly designed for knowledge-based intranet portal (IEX4.0, running on SharePoint 2013) with these objectives:
  • Have a highly intuitive and efficient platform for IE officers using the intranet for their knowledge work;
  • Ensure knowledge assets are inter-linked to provide a more logical view of knowledge within IEX;
  • Have business relevant features that will help increase workgroup collaboration within IEX;
  • Enable high search relevancy including federated search from various sources (e.g. CRM System);
  • Have a highly intuitive mobile responsive website, thereby eradicating the need for a native mobile application;
  • Optimize out-of-the-box features from SharePoint 2013
  • Provide a scalable intranet for ease of maintenance and migration;
  • Minimize customization
Responsibility
  • Took part in Requirement Gathering, working with customer and Business Analyst for technical solution.
  • Proposed and designed the system, wrote technical design.
  • Proposed, setup and configured SharePoint small purpose farm server with 4 servers play different role: 1 Front-End server, 1 Application Server, 1 Database server, 1 Database Server and 1 Standalone server for Office Web Apps.
  • Configured and maintained Content Sources, Crawl Rules, File Types, Crawl properties and Managed Metadata within the Search Service Application.
  • Deployed Custom Solution Packages (WSP) using PowerShell commands.
  • Reviewed log files in ULS Viewer to perform troubleshooting and resolve issues and errors in a timely fashion.
